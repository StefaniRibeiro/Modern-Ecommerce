# Modern E-commerce

Aplicação de e-commerce moderna desenvolvida com foco em boas práticas de arquitetura front-end e experiência do usuário.

O projeto foi construído utilizando tecnologias modernas do ecossistema React, com o objetivo de simular uma interface real de loja virtual.

## 🚀 Tecnologias

* Next.js
* TypeScript
* Tailwind CSS
* Zustand
* shadcn/ui

## 📦 Funcionalidades

* Listagem de produtos
* Página de detalhes do produto
* Carrinho de compras
* Layout responsivo
* Estados de carregamento (loading)
* Persistência do carrinho

## 🔗 API

Os produtos são obtidos através de uma API pública para fins de demonstração.

## 🖥️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/modern-ecommerce.git
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
